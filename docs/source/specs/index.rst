History
=======


Original app have been developped by `Vanoise National Park <http://www.vanoise-parcnational.fr>`_

`PnVanoise/cables <https://github.com/PnVanoise/cables>`_

In order to improve app, based on old technologies, this new version have been entirely newly developped.

Technologies
============

Database
++++++++

* PostgreSQL
* PostGIS

Backend
+++++++

* Django 
* GeoDjango
* Django Rest Framework

Frontend
++++++++

* Nuxt (Vue JS framework) in SPA/PWA mode
* Vuetify (UI framework)
* LeafletJS
